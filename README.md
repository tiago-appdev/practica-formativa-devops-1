### Descripción del proyecto

El proyecto consiste en una aplicación web que permite a los usuarios registrarse y acceder a un sistema de gestión de tareas. Los usuarios pueden crear, editar y eliminar tareas, así como marcarlas como completadas. La aplicación utiliza una base de datos para almacenar la información de los usuarios y las tareas.
